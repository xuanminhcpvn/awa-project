## 4. Installation Instruction

Instruction to install and use the web application is written below step by step:

1. Clone the GITHUB repository to local machine.  
   Go to the folder you want the git repository to reside in, then run this command:

```bash
git clone https://github.com/xuanminhcpvn/Introduction-to-Web-Programming-Project.git
```
Go to the cloned folder:
```bash
cd CT30A3204-Advanced-Web-Applications-Final-Project
```
Run the following npm commands in order:
```bash
npm run install
npm run build   # builds dist folder
npm run start   # starts server and client
```
Finally, open the application in your browser:
```bash
http://localhost:1234
```
